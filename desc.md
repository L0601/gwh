    我要给女朋友做一个桌面程序，主要是管理自己作为编辑的接单记录，即她会和一些出版社合作，帮忙校稿，然后回去收益。
需要有一下功能：
1. 支持现有数据导入，且需要明确导入的格式
2. 支持系统数据导出
3. 对接人管理，新增、删除对接人，新增编辑任务时需要可以选择对接人
4. 新增、修改编辑任务，包括一下：
    1. 接单日期，完成日期，打印日期（可选）、费用（可选）、是否结账（），结账金额（联动是否结账），编辑书名，页码可选，字数可选，是否完成（结账后自动完成，也可以手动完成），对接人
    2. 分3个类别展示，进行中，完成未结账，已结账，可以按照接单日期、对接人筛选
    3. 新增统计模块
    按照日期（默认最近一个月、可选择日期范围）、接单人（下拉可选可空）展示接单数目、累计已收金额 接单详情，可跳转到对应的接单详情
    4. 支持录入编辑任务，修改任务状态，完善报酬金额等操作
技术栈
Vue + Tauri + SQLite 支持 windows 和 mac 系统
要求：
    页面简洁美观
    交互人性化

添加成功过或者失败需要有toast
值校验不通过 提交时回显到输入框
时间显示精确到分钟就可以了
